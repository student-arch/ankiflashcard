Import Instructions

  1. Open Anki Desktop → File → Import
  2. Select the downloaded CSV file
  3. Set "Allow HTML in fields" to Yes
  4. Map Field 1 → Front, Field 2 → Back, Field 3 → Image_Prompt, Field 4 → Tags, Field 5 → Notes
  5. Click Import and you're done!

Quick Start

  • Choose a content source tab
  • Add your OpenRouter API key
  • Click "Create Flashcards"
  • Download CSV & import to Anki

API Setup

OpenRouter (Recommended)

  1. Visit openrouter.ai
  2. Create account & get API key
  3. Use free model: openrouter/free
